Printing to the Console

One of the first things you'll learn in any programming language is how to print output to the console. In Java, you use the System.out.println method to do this.

              
 public class HelloWorld {
    public static void main(String[] args) {
       System.out.println("Hello, World!");
       }
       }

When you run this code, it will output:

            Hello, World!
