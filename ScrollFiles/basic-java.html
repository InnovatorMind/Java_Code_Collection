Printing to the Console

One of the first things you'll learn in any programming language is how to print output to the console. In Java, you use the System.out.println method to do this.

              
public class HelloWorld {
   public static void main(String[] args) {
      System.out.println("Hello, World!");
      }
 }

When you run this code, it will output:

            Hello, World!

1. Printing Normal Text

Example-

              
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Java is awesome!");  
    System.out.println("Learning Java step by step.");  
    System.out.println("12345");  
    }
}

You can print any text string by enclosing it within double quotes ("). The text will be displayed exactly as it appears.

2. Printing Variables

Example-

              
public class HelloWorld {
  public static void main(String[] args) {
    int number = 10;
    System.out.println(number);
      }
    }

You can print the value of variables. Just pass the variable name to System.out.println(), and it will print the current value of that variable
more Example- 

              
String name = "Alice";
System.out.println(name);  // Prints: Alice

double pi = 3.14159;
System.out.println(pi);  // Prints: 3.14159

boolean isJavaFun = true;
System.out.println(isJavaFun);  // Prints: true

3. Print with Concatenation

Example-

              
public class HelloWorld {
  public static void main(String[] args) {
    String greeting = "Hello";
    System.out.println(greeting + ", World!");
    }
}

You can concatenate strings and variables using the + operator to create a more complex output. More Examples:

              
int age = 25;
System.out.println("I am " + age + " years old."); 
 // Prints: I am 25 years old.
String firstName = "John";
String lastName = "Doe";
System.out.println("Name: " + firstName + " " + lastName);  
// Prints: Name: John Doe

4. Print Special Characters

Example-

              
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello\nWorld!");
    }
}

Special characters like \n, \t, etc., can be used to format the output.

  • \n: Newline (moves to the next line)
  • \t: Tab (inserts a horizontal tab space)
  • \\: Backslash (prints a single backslash)
  • \": Double quote (prints a double quote)

              
System.out.println("Java\nProgramming");  
System.out.println("1\t2\t3"); 
System.out.println("Path: C:\\Program Files\\Java"); 
System.out.println("She said, \"Java is fun!\"");

5. Print Expressions

Example-

              
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println(5 + 3);
    }
}

You can print the result of arithmetic or logical expressions directly.
More Example:

            
System.out.println(10 * 2); 
System.out.println(15 / 3);  
System.out.println(7 % 2); 
System.out.println(5 > 3);
